# SE Tool — Installation Guide

## Requirements
- Windows 10 or later
- No Python or software installation needed

---

## Steps

### 1. Open the shared drive folder

Navigate to:
```
Z:\SIPC\Software\SE_TC_Tool
```

### 2. Run the installer

Double-click **`install.bat`**

> If Windows shows a security warning, click **"Run anyway"**

### 3. Choose shortcut location

The installer will ask:

```
Where would you like to save the shortcut?

 [1] Desktop
 [2] Custom location

Enter 1 or 2:
```

Type **`1`** and press **Enter** to add it to your Desktop.

### 4. Done

A shortcut named **SE Tool** will appear on your Desktop.
Double-click it to launch the app — it will open automatically in your browser.

---

## Notes

- The app runs locally on your machine. No internet connection required.
- To uninstall, simply delete the Desktop shortcut.
- If the app does not open after a few seconds, check that your browser is not blocking `localhost`.
